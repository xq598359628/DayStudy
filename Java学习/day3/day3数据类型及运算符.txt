1标识符的规则

	标识符:
		用于给程序中变量、类、方法命名的符号。
	规则：
		1由字母数字下划线和美元符$组成。不能是数字开头
		2不能是保留字和关键字，但可以包含。
		3不能包含空格
		4不能包含除美元$以外的特殊符号
		
		包名：小写 
		类或接口 大写
		方法或变量 首字母小写
		常量 大写
		
2Java关键字

	关键字：
		具有特殊用途的单词。
		所有的关键字都是小写的。Java共有50个关键字。 enum是java5新增。goto 和const被称为保留字。
	保留字：Java现在还没使用这两个关键字，但以后可能使用
	直接量：true,false,null
	
3Java是强类型还是弱类型语言

	Java是强类型语言。两个含义：1变量必须先声明后使用 2变量只能接受与之类型相匹配的值
	
4编程及变量的本质。
	
	编程的本质就是对内存中数据的访问与修改。程序所用的数据都保存在内存中，程序员需要一种机制来访问或修改。
	这种机制就是变量。每个变量都代表了内存中的某一小块内存。

	基本类型和引用类型
	基本类型： 
				整数类型 
						byte 1字节  -2^7 ~ 2^7-1  
						short 2字节 -2^15 ~-2^15-1
						int 4字节 -2^31 ~ 2^31-1
						long 8字节 -2^63 ~ 2^63-1
				字符类型
						char 2个字节
				浮点类型
						float 4个字节
						double 8个字节
				布尔类型
						boolean
	注意事项：
			1在定义Long或者Float类型变量的时候，要加L或者F。
			2byte short 在定义的时候，他们接收的其实是一个int类型的型。因为系统默认整数值是Int，浮点型式double
			3char类型可以直接当作整型来使用，相当于一个无符号16位整数，0-65536
6float和long的存储大小问题

		1它们的底层的存储结构不同
		2float范围比long大		
	
6char可以来存储汉字么？
	char型变量是用来存储Unicode编码的字符的，unicode编码字符集中包含了汉字，所以，char型变量中可以存储汉字。不过，如果某个特殊的汉字没有被包含在unicode编码字符集中，那么，这个char型变量中就不能存储这个特殊汉字。
	补充说明：unicode编码占用两个字节，所以，char类型的变量也是占用两个字节。
	
7自动类型转换与强制类型转换

	在一个运算过程中，有一个大的数据类型，小的数据类型自动向大的数据类型转换
		A:byte,short,char—int—long—float—double
		B:byte和short, char相互之间不转换，他们参与运算首先转换为int类型
	两种特殊情况
		1如果直接将一个较小的整数(在byte short的数值范围内)赋值给byte short，系统自动将这个整数值当作byte,short
		2如果将一个整数值负值给Long，使用L
